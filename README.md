🧭 About Jobhunter
Welcome to Jobhunter, a full-stack MERN job portal project that connects job seekers and employers through an intelligent, AI-enhanced recruitment experience.

Whether you’re a developer looking for a GitHub project with AI integration, or an organization exploring how AI can enhance hiring workflows, Jobhunter is a modern example of integrating AI + MERN stack effectively.

🔍 Why Jobhunter?
Jobhunter isn’t just another job portal — it’s a smart recruitment ecosystem that leverages OpenAI’s API to automatically generate personalized job descriptions, helping companies attract better candidates faster.

🧩 Specializations
👩‍💼 For Job Seekers
Search and filter jobs based on skills, location, and experience.
Manage your profile, skills, and work experience intuitively.
Track all your applications and saved jobs from a single dashboard.
🏢 For Employers
Create and manage job postings with ease.
Use AI to generate compelling job descriptions automatically.
Track applicants, shortlist candidates, and manage your recruitment funnel.
🤖 AI-Powered Job Description Generator
Jobhunter uses OpenAI’s GPT-based API to generate SEO-optimized, personalized job descriptions.
This makes hiring faster, improves job post visibility, and attracts high-quality applicants — a feature that sets Jobhunter apart from traditional job portals.

⚙️ Features
✅ User Authentication — Secure signup & login using JWT.
✅ AI Job Description Generator — Auto-generate optimized job posts using OpenAI.
✅ Job Search & Filtering — Advanced filters for job type, category, and location.
✅ Profile Management — Upload profile photos, add skills, education, and work history.
✅ Employer Dashboard — Manage listings, applicants, and job analytics.
✅ Application Tracking System (ATS) — View, update, and manage job applications.
✅ Company Management — Maintain company profiles and recruit effortlessly.
✅ Cloud Image Uploads — Seamless media uploads with Cloudinary.
✅ Responsive UI — Fully optimized for mobile, tablet, and desktop.

🧠 Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
State Management: Redux Toolkit
Authentication: JWT (JSON Web Token)
Image Handling: Cloudinary, Multer
AI Integration: OpenAI API
Deployment:

Frontend → Vercel / Cloudflare
Backend → AWS EC2 + Nginx + PM2
CI/CD: GitHub Actions
